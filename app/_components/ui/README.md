# UI Components

This directory contains all the base UI components used throughout the application. These components are designed to be reusable, consistent, and follow the design system defined in the styles directory.

## Directory Structure

- `/badges`: Badge components for labels and status indicators
- `/buttons`: Button components with various styles and variants
- `/cards`: Card components for displaying content
- `/form`: Form input components like text inputs, selectors, and controls
- `/icons`: Icon components and wrappers
- `/layout`: Layout components like grids, sections, and containers
- `/links`: Link components for navigation
- `/lists`: List components for displaying collections of items
- `/navigation`: UI components used in navigation (tabs, etc.)
- `/toggle`: Toggle and switch components
- `/typography`: Text components with various styles

## Usage Guidelines

1. Always use these UI components instead of creating new ones for similar purposes
2. Follow the type definitions from `/app/_types/ui.ts` when extending components
3. Maintain consistency with the design system (colors, typography, spacing)
4. Keep components small, focused, and composable
